# STORWELL Embedded Software Engineer

This repo is for appling the embedded software engineer position at storwell

## Install command

make all

## Useage

./calcPaper {N} {M}
PS: N and M must be positive integers

example:

./calcPaper 6 5
5x5, 1x1, 1x1, 1x1, 1x1, 1x1

./calcPaper 1 1
1x1

./calcPaper 9 9
9x9
